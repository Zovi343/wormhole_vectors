# Wormhole Vectors

Interactive 3D demo: **Sparse** and **Dense** vector spaces (rotating stacked planes, document points, dimension arrows), **Graph** space (3D node–link network), and a sequential purple **wormhole** beam (Sparse → Dense → Graph).

## Run locally

```bash
cd wormhole_vectors
npm install
npm run dev
```

Then open the URL Vite prints (usually `http://localhost:5173`).

## Build

```bash
npm run build
npm run preview   # optional: serve dist/
```

## Controls

- **Restart Demo** — reset the animation timeline.
- **Pause / Resume** — toggle the render loop.
